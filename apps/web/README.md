# WorkIndex web

The candidate-facing WorkIndex MVP built with Next.js 16 App Router, React 19, TypeScript, and Tailwind CSS 4.

## Commands

```bash
pnpm dev
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```

The current records are fictional, visibly labelled product fixtures. Do not replace the demo label until records come from the compliance-gated ingestion and review pipeline.
